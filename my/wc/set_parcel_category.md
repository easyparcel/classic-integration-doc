# Set parcel category for order

> ⚠️ **Parcel Category is required for international order, EasyCover and DDP.**

### Step 1: Set Parcel Category

In **Woocommerce**, you can set it at the order fulfilment page.

<img width="1116" height="614" alt="image" src="https://github.com/user-attachments/assets/14b8858e-7f71-42ec-9c3f-038d8efbafaf" />

### Step 2: Select Parcel Category

You can click on the **Parcel Category** to select the parcel category that related to your order.

<img width="1119" height="617" alt="image" src="https://github.com/user-attachments/assets/74fef8b4-2bb3-4ec0-ba30-f3598c09c51b" />

### Step 3: Save Parcel Category

After selected, you are able to save the Parcel Category by click the **Save Parcel Category** button 

<img width="1118" height="600" alt="image" src="https://github.com/user-attachments/assets/a421ac07-4159-45ab-98c6-9bc83e0aa096" />
