# How to set HS code for order

## To set the HS Code for an order, you’ll need to assign the HS code at the product level. 

> ⚠️ **HS code is required for international order.**

### Step 1: Set HS code in product

In **Shopify**, you can do this by going to **Products**, selecting the product you want to update, and then scrolling down to the **Shipping** section on the product details page. Under **Customs information**, you’ll be able to enter the HS code.

<img width="1280" height="670" alt="image" src="https://github.com/user-attachments/assets/ee3e7750-0c53-41e0-a537-04a8648452aa" />

### Step 2: Create order with the product

When you create an order with the product that has set the HS code, the order will show the HS code of the product.

<img width="624" height="657" alt="image" src="https://github.com/user-attachments/assets/3415ae81-b795-4d36-84a4-1edefd7e5f57" />

### Step 3: Change HS code at fulfilment page

You will also able to change the HS code on the fulfilment page. After changing the HS code, you will need to click the **Save** button.
